### Installation
```shell
# Windows WSL2 (Ubuntu)
> dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
> dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
> wsl --list -o
> wsl --install -d ubuntu

# WSL2 Ubuntu with Docker & K8s Installation
$ sudo snap install docker
$ sudo groupadd docker
$ sudo usermod -aG docker $USER
$ docker --version
$ [ $(uname -m) = x86_64 ] && curl -Lo ./kind https://kind.sigs.k8s.io/dl/v0.27.0/kind-linux-amd64
$ sudo chmod +x ./kind
$ sudo mv ./kind

# docker images
$ docker pull hello-world:latest
$ docker pull busybox:latest
$ docker pull nginx:latest
```

### Container Networks
- Bridge mode *(default)*
  ```shell
  $ tcpdump -i docker0 -n
  
  # tty1
  $ docker run -it --name=box1 busybox
  $ ip addr
  $ ping -c 4 172.17.0.3
  
  # tty2
  $ docker run -it --name=box2 busybox
  $ ip addr
  $ ping -c 1 8.8.8.8
  ```

- Host mode
  ```shell
  # 컨테이너 실행
  $ docker run --rm -d --network host --name nginx nginx:latest
  $ docker inspect my_nginx
  $ netstat -ntlp
  $ curl --head localhost
  ```

  <img src="{{"/assets/img/04195.png" | relative_url}}" alt="" style="">

- None mode
  ```shell
  $ docker run --rm -d --network none --name my_nginx nginx
  $ docker inspect nginx | grep NetworkMode
  $ netstat -ntlp
  $ docker exec -it nginx /bin/bash
  $ ip addr
  $ ping 8.8.8.8
  $ curl localhost
  ```
